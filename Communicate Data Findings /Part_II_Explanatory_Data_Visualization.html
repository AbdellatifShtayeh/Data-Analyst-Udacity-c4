Part II - Insights from Prosper Loan Data¶

by Abdellatif Shtayeh

Investigation Overview¶

This presentation aims to highlight key trends and relationships in the Prosper Loan Data, focusing on loan performance, borrower rates, and borrower characteristics. The goal is to provide clear, actionable insights for stakeholders.

Executive Summary:

  • Higher borrower rates are associated with increased default rates.
  • Prosper rating is a strong predictor of loan outcomes.
  • Borrower income influences loan amounts and rates.

Dataset Overview and Executive Summary¶

The Prosper Loan Data dataset includes 113,937 loans with 81 variables, such as loan amount, borrower rate, loan status, and borrower income. This analysis distills the data into key findings:

  1. Borrower rates peak around 0.15-0.20.
  2. Higher rates correlate with defaults.
  3. Prosper rating effectively indicates loan risk.

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
%matplotlib inline

# Suppress warnings
import warnings
warnings.simplefilter('ignore')

# Load the dataset
df = pd.read_csv('Data/prosperLoanData.csv')

# Define the order for LoanStatus
loan_status_order = ['Current', 'Completed', 'FinalPaymentInProgress', 'Past Due (1-15 days)', 'Past Due (16-30 days)', 'Past Due (31-60 days)', 'Past Due (61-90 days)', 'Past Due (91-120 days)', 'Chargedoff', 'Default']

# Convert LoanStatus to ordered categorical type
df['LoanStatus'] = pd.Categorical(df['LoanStatus'], categories=loan_status_order, ordered=True)
